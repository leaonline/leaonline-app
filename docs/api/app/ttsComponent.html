Tts stands for Text-To-Speech. It contains an icon and the text to be spoken.

Parameters:
Name Type Attributes Description
props.text: string

The displayed and spoken text
props.ttsText: string

The spoken text, use this if it differs from written text
props.dontShowText: boolean

Determines whether the text is displayed (Default 'true')
props.smallButton: boolean

Changes the button size from 20 to 15 (Default 'false')
props.block: boolean <optional>

Makes the container flexGrow. If this causes problems, use style instead.
props.disabled: boolean <optional>

Makes the button disabled
props.color: string <optional>

The color of the icon and the text, in hexadecimal format. Default: Colors.secondary (examples in ./constants/Colors.js)
props.iconColor: string <optional>

The color of the icon in hexadecimal format. Default: Colors.secondary (examples in ./constants/Colors.js)
props.activeIconColor: string <optional>

The color of the icon when speaking is active
props.shrink: number

The parameter to shrink the text. Default: 1
props.fontSize: number

The parameter to change the font size of the text. Default: 18
props.fontStyle: string

The parameter to change the font style of the text. Default: 'normal' ('italic')
props.style: object <optional>

The parameter to change the font style of the text. Default: 'normal' ('italic')
props.align string

Defines the vertical alignment of the button and text
props.paddingTop: number

Determines the top padding of the text. Default: 8
props.speed: number

Determines the speed rate of the voice to speak. Default: 1.0

The parameter to identify the buttons
