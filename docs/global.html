# UnitScreen(props) → {JSX.Element}

On this screen, the respective Unit is displayed and the users can interact with it, by solving the tasks on it's pages.

If a unit is completed and there is no next unit in the queue, it navigates the users to the {CompleteScreen}.

If a next unit exists, it will load this next unit.

On cancel, it navigates users back to the {MapScreen}.
