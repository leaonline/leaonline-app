# constant callMeteor

Wraps a Meteor.call in a promise but also allows to hook in various stages of the request. Always returns a promise.

Meteor calls are rpc-style calls to an endpoint in our Meteor server.

If failure callback is provided, it will automatically server as the catch argument of the promise.

Automatically resolves to {failure} when there is no active connection to the Meteor server.
